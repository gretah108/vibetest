# Personal Site

`spec.md`를 기준으로 만든 개인 홍보용 원페이지 사이트입니다.

## 파일 구성

* `index.html` : 페이지 구조
* `style.css` : 전체 스타일과 반응형 레이아웃
* `script.js` : 텍스트 데이터와 카드 렌더링
* `assets/` : 프로필/포트폴리오/취미 이미지 폴더

## 수정 방법

가장 먼저 수정할 파일은 `script.js`입니다.

상단의 `siteContent` 객체에서 아래 내용을 한 번에 바꿀 수 있습니다.

* 이름
* 소개 문구
* 이력
* 포트폴리오 4개
* 취미 3개
* 이미지 경로

## 이미지 교체

아직 실제 이미지가 없어도 사이트는 플레이스홀더로 표시됩니다.

원하는 사진을 아래 이름으로 넣으면 자동으로 반영됩니다.

* `assets/profile.jpg`
* `assets/portfolio1.jpg`
* `assets/portfolio2.jpg`
* `assets/portfolio3.jpg`
* `assets/portfolio4.jpg`
* `assets/hobby1.jpg`
* `assets/hobby2.jpg`
* `assets/hobby3.jpg`

## 실행 방법

`index.html`을 브라우저에서 열면 바로 확인할 수 있습니다.
